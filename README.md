# 🍞 Калькулятор ХЕ

[![.NET](https://img.shields.io/badge/.NET-10.0-512BD4?logo=.net&logoColor=white)](https://dotnet.microsoft.com/)
[![Blazor](https://img.shields.io/badge/Blazor-WebAssembly-512BD4?logo=blazor)](https://dotnet.microsoft.com/apps/aspnet/web-apps/blazor)
[![Bootstrap](https://img.shields.io/badge/Bootstrap-5.3-7952B3?logo=bootstrap)](https://getbootstrap.com/)

Экспериментальный проект на Blazor WebAssembly. Рассчитывает угеводный коэффициент (ХЕ на 1 ед. инсулина) и выводит таблицу соответствия доз инсулина и ХЕ с шагом 0.5.
